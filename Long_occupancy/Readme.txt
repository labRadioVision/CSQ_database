Each folder contains measurements for a given device (i.e., Nexus_cellphoneC, Samsung_TabletB, Xperia_cellphoneA and Xperia_cellphoneD) 
in a given date (YYYY-MM-DD). It also contains the true_occupancy file. 

true_occupancy file: date(YYYY-MM-DD),starttime(tA) (HH-MM),endtime(tB) (HH-MM), distance from body for devices A,B,C,D. 

for a given device and date each folder contains four files: 
CID:cell indentifier corresponding to a given sample
CSQ:received signal strength corresponding to a given sample
DATE:date of sampling (YYYY-MM-DD)
TIME:time of sampling (HH-MM-ss)
