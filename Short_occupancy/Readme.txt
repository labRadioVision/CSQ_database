Short Occupancy CSQ Data

Each folder contains measurements for a given day (i.e., DAY_1, DAY_2, DAY_3) for Nexus_cellphoneC. 
It also contains the true_occupancy file. 

true_occupancy file: starttime(tA) (HH-MM),endtime(tB) (HH-MM). 

For a given device and date each folder contains three files: 

LCID: cell indentifier corresponding to a given sample
CSQ:received signal strength corresponding to a given sample
TIME:time of sampling (HH-MM-ss)
